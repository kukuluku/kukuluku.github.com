# kubectx와 kubens
[kubectx + kubens: Power tools for kubectl](https://github.com/ahmetb/kubectx)

## Install
[krew](https://github.com/kubernetes-sigs/krew/)를 사용하여 kubectl 의 plugin으로 설치한다. 이 방식으로 설치하면 ```kubectl ctx``` 와 ```kubectl ns``` 형태로 사용한다.<br/>
만약 shell completion scripts 로 사용하려면 ```Homebrew```나 ```apt``` 등의 패키지 매니저로 직접 설치해야 한다.

```
# kubectl plugins(macOS & Linux)
kubectl krew install ctx
kubectl krew install ns

# Homebrew
brew install kubectx
brew install kubens
```

## kubectx
다중 클러스터 환경에서 ```kubectl```로 context를 편리하게 전환하기 위한 툴
```
kubectx <k8s-cluster-context-name>
# kubectx 명령어 실행 시 목록에서 context를 선택할 수 있음
```

## kubens
```kubectl```에서 클러스터의 namespace를 매번 지정하지 않고 고정으로 지정할 수 있도록 하는 툴
```
kubectx <k8s-namespace-name>
# kubens 명령어 실행 시 목록에서 namespace를 선택할 수 있음
```
